# Hotel Management System

## Overview
A Hotel Management System developed using Java, JDBC, and MySQL.

## Features
- Customer Registration
- Customer Login
- View Available Rooms
- Book Room
- View Bookings
- Cancel Booking
- Search Room
- Update Room
- Delete Room

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE
- Git
- GitHub

## Database
Database Name: hoteldb

Tables:
- customers
- rooms
- bookings

## Project Structure

```
src/
├── dao
├── model
├── util
└── main
```
